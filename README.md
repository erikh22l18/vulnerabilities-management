# Project Vulnerability Management

This project is a web application designed for managing vulnerabilities within organizations and their associated projects. It provides tools for tracking, assigning, and resolving security vulnerabilities.

## Features

*   **Organization Management:** Manage multiple organizations and their users.
*   **Project Management:** Create and oversee projects within each organization.
*   **Task Management:** Assign tasks related to vulnerabilities to specific users.
*   **Vulnerability Tracking:** Log, categorize, and monitor the status of identified vulnerabilities.
*   **User Roles and Permissions:** Control access and actions based on user roles.
*   **Reporting:** (Potentially) Generate reports on vulnerability statuses and trends.

## Tech Stack

*   PHP / Laravel
*   Blade for templating
*   Tailwind CSS for styling (likely)
